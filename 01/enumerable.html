1 Enumerable

Si tenemos una clase que implementa el método each(&block), al incluir en ella el módulo enumerable, éste nos brinda una serie de métodos que lo utilizan para realizar distintas acciones sobre una colección.

A continuación una lista de dichos métodos:

1.1 all?

Pasa cada uno de los elementos al bloque. El método devuelve verdadero si el bloque nunca retorna false o nil y falso si alguno el bloque devuelve false o nil en alguna de las iterciones.

El bloque es opcional y si no se provee, el método aporta un bloque: 

[...].all? {|o| o}
